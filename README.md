## About The Project

This is my pet-project that lets you see news form [NewsAPI](https://newsapi.org/)

### Built With

What is used in this project:
* Kotlin
* DaggerHilt
* Paging3
* MVVM



<!-- USAGE EXAMPLES -->
## Usage

NewsApp lets you:
- See trending news in specific country.
- Watch news from all over the world.
- Search news.
